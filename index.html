What I do

Below is a list of tools I have picked up along my path as a software engineer. I enjoy the entire process of software development, and as such you will see a variety of tools ranging in use from the back to front end.

React

My prefered front end framework would be React.js. I find the power of functional components to be unmatched.

Redux

A single source of truth on the front end can be an invaluable tool for making applicaitons predictable.

JavaScript

My favorite programming language! It might be a bit quirky, but I like it that way.

Python

Sometimes you need the right tool for the job. Fortunatly Python has a big tool-box and a lot of open source supporters that keep that tool box stacked.

HTML

Hyper Text Markup Language, when you need to talk to web browsers there's really only one option.

CSS

Cascading Style Sheets, because vanilla HTML is yucky!

PSQL

Relational databases are a must, and a free open-source Relational Database Management System has come in handy more than once.

Git

Distributed version-control, so that whey you accidentally wipe out a project you have the pleasure of also finding out the last time you committed was two days ago..

Express

The de facto backend framework when working with Node.js.

Flask

If you want to spin up an API with Python, Flask has got your back.
